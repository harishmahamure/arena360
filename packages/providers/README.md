# @gaming-cafe/providers

Shared React provider stack: theme, toast, error boundary, query client. The composed `<AppProviders />` and the individual exports land in a later phase; this package is an empty skeleton today so the workspace graph compiles.

## Peer dependencies

- `react@^19`
- `@mui/material@^7`
- `@gaming-cafe/theme` (workspace)
- `react-toastify@^10`
