# @gaming-cafe/ui

Shared MUI-based React components. The public surface (per `docs/ARCHITECTURE.md`) lands in later phases: `<DataGrid />`, `<FormBuilder />`, `<PageLayout />`, `<ConfirmDialog />`, and toolbar primitives.

For now this package is an empty skeleton so the Turbo / TypeScript graph compiles. Do not add component code here until the relevant migration phase is unblocked.

## Peer dependencies

- `react@^19`
- `@mui/material@^7`
- `@gaming-cafe/theme` (workspace)
